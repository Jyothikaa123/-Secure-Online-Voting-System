Online Voting System

A secure and user-friendly Online Voting System built using Python Flask, HTML, CSS, and JSON. The system provides separate dashboards for Admins, Candidates, and Voters, allowing election management, candidate enrollment, secure voting, and result tracking.

🚀 Features
👨‍💼 Admin Module
Admin Registration and Login
Create and Manage Elections
Set Election Type, Name, ID, Start Time, and End Time
View Election Results
Monitor Candidates and Votes
🧑‍💻 Candidate Module
Candidate Registration and Login
Election Enrollment using Election Name and Election ID
View Participated Elections
Track Votes Received
View Election Results
🗳️ Voter Module
Voter Registration and Login
Search Elections using Election Name and Election ID
View Ongoing Elections
Cast Votes Securely
Prevent Multiple Voting in the Same Election
View Previous Election Results
🛠️ Technologies Used
Python
Flask
HTML5
CSS3
JavaScript (Optional)
JSON (Data Storage)
